# Lucky 21 (blackjack-reel) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace Hit or Bust **in the `blackjack-reel` family** with **Lucky 21** — a stop-the-reels press-your-luck card slot: five spinning reels stopped one at a time (honest payline read), an honest shuffled 52-card deck dealt without replacement, five reel *roles* (deal / lock-in bonus / mix / big), additive multipliers, −2/−3 minus cards, two bust conditions (over 21 or a BUST symbol), a qualify-at-15 ratcheting payout, Five-Card Charlie ×3 with a qualification floor, and exact RTP under optimal stopping with deck depletion — keeping the family id, all dispatch seams, and the 10th floor slot.

**Architecture:** The `blackjack-reel` family already exists and is threaded through every `never` switch, so this is an **in-place replacement** of the family's internals, not a new family. Pure engine modules do the work: a new `deck.ts` (52-card build + Fisher–Yates shuffle + value lookup), a rewritten `blackjackReel.ts` (hand evaluation with minus + the stop-reel step functions that mutate `MachineSessionState.blackjackReel`), and a rewritten `blackjackReelRtp.ts` (deck-depletion backward-induction DP computing the optimal cash/continue policy and exact RTP). The def/state types evolve in place; `simulateMachine` and `exactRtp` re-point to the new flow; the store gets stop/cash actions; the reel surface + result modal + chrome are reworked. Calibrated to ~90% RTP *using the DP itself*, then frozen.

**Tech Stack:** Nuxt 4, Vue 3 `<script setup>` + TS, Pinia, Vitest + @vue/test-utils + happy-dom, the v0.7.0 chrome system.

## Card & symbol model (used throughout)

- **Cards are SymbolIds** `"<rank><suit>"`: ranks `A 2 3 4 5 6 7 8 9 T J Q K`, suits `S H D C` → e.g. `AS`, `TH`, `KD` (52 ids). Built by `deck.ts`, **not** listed in `def.symbols` (rendered by a Card component, not the icon registry).
- **Specials are SymbolIds** in `def.symbols`: multipliers `MX2 MX3 MX5 MX10`, minus `MM2 MM3`, bust `BUST`.
- **Reel composition token** `CARD` (reserved sentinel SymbolId): in `def.reels`, a `CARD` slot is filled at deal time with the next card off the shuffled deck; any other token is a literal special.

```ts
import type { RandomFn } from './rng'
import type { SymbolId } from './types'

const RANKS = ['A', '2', '3', '4', '5', '6', '7', '8', '9', 'T', 'J', 'Q', 'K'] as const
const SUITS = ['S', 'H', 'D', 'C'] as const

export function buildDeck(): SymbolId[] {
  const d: SymbolId[] = []
  for (const s of SUITS) for (const r of RANKS) d.push(`${r}${s}`)
  return d
}
export function shuffle(cards: SymbolId[], rand: RandomFn): SymbolId[] {
  const a = [...cards]
  for (let i = a.length - 1; i > 0; i--) {
    const j = Math.floor(rand() * (i + 1))
    ;[a[i], a[j]] = [a[j]!, a[i]!]
  }
  return a
}
export function cardRank(id: SymbolId): string { return id.slice(0, -1) }
export function cardSuit(id: SymbolId): 'S' | 'H' | 'D' | 'C' { return id.slice(-1) as 'S' | 'H' | 'D' | 'C' }
export function cardColor(id: SymbolId): 'red' | 'black' { const s = cardSuit(id); return s === 'H' || s === 'D' ? 'red' : 'black' }
export function isAce(id: SymbolId): boolean { return cardRank(id) === 'A' }
export function cardValue(id: SymbolId): number {
  const r = cardRank(id)
  if (r === 'A') return 11
  if (r === 'T' || r === 'J' || r === 'Q' || r === 'K') return 10
  return Number(r)
}
```

```ts
import type { BlackjackReelMachineDef, SymbolId } from './types'
import { cardValue, isAce } from './deck'

export interface HandAcc { hard: number, aces: number, multSum: number }
export function freshAcc(): HandAcc { return { hard: 0, aces: 0, multSum: 0 } }
type EvalCfg = Pick<BlackjackReelMachineDef, 'multiplierSymbols' | 'minusSymbols' | 'bustSymbol'>

/** Fold one symbol into the accumulator. BUST is never passed here (step fn handles loss). */
export function applySymbol(acc: HandAcc, def: EvalCfg, sym: SymbolId): HandAcc {
  if (sym in def.multiplierSymbols) { acc.multSum += def.multiplierSymbols[sym]!; return acc }
  if (sym in def.minusSymbols) { acc.hard = Math.max(0, acc.hard - def.minusSymbols[sym]!); return acc }
  if (isAce(sym)) { acc.aces++; acc.hard += 1; return acc }
  acc.hard += cardValue(sym)
  return acc
}
/** Best total ≤ 21 (promote one ace to 11 if it fits), and the all-hard "soft low". */
export function bestTotal(hard: number, aces: number): { total: number, isSoft: boolean, softLow: number } {
  if (aces > 0 && hard + 10 <= 21) return { total: hard + 10, isSoft: true, softLow: hard }
  return { total: hard, isSoft: false, softLow: hard }
}
export interface HandEval { total: number, isSoft: boolean, softLow: number, busted: boolean, multSum: number }
export function evaluateHand(def: EvalCfg, syms: readonly SymbolId[]): HandEval {
  const acc = freshAcc()
  for (const s of syms) applySymbol(acc, def, s)
  const bt = bestTotal(acc.hard, acc.aces)
  return { total: bt.total, isSoft: bt.isSoft, softLow: bt.softLow, busted: bt.total > 21, multSum: acc.multSum }
}
```

```ts
import type { MachineSessionState, SpinOutcome } from './types'
import type { RandomFn } from './rng'
import { buildDeck, shuffle } from './deck'

export function payEntry(paytable: { total: number, pay: number }[], total: number): number {
  return paytable.find(e => e.total === total)?.pay ?? 0
}

/** Best-total paytable × additive multiplier × ante, with the qualify floor, natural premium,
 *  and the Charlie multiplier + qualification floor. Returns whole credits. */
export function handPayout(def: BlackjackReelMachineDef, bj: BlackjackReelSessionState): number {
  if (bj.busted) return 0
  const mult = Math.max(1, bj.multSum)
  // Charlie guarantees qualification: floor the base at the qualify rung.
  let base = bj.charlie
    ? Math.max(payEntry(def.paytable, bj.bestTotal), payEntry(def.paytable, def.qualifyMin))
    : payEntry(def.paytable, bj.bestTotal)
  if (bj.natural && bj.bestTotal === 21) base = def.naturalPay
  const charlieMul = bj.charlie ? def.charlieMultiplier : 1
  return base * mult * charlieMul * bj.ante
}

export function dealReels(def: BlackjackReelMachineDef, state: MachineSessionState, coins: number, rand: RandomFn): SpinOutcome {
  const bj = freshBlackjackState(); bj.ante = coins; bj.phase = 'spinning'
  const deck = shuffle(buildDeck(), rand); let di = 0
  bj.reelStrips = def.reels.map(slots => slots.map(tok => (tok === 'CARD' ? deck[di++]! : tok)))
  state.blackjackReel = bj
  return outcome(def, bj, coins, coins, [{ type: 'cards-dealt', strips: bj.reelStrips.map(s => [...s]) }])
}

export function stopReel(def: BlackjackReelMachineDef, state: MachineSessionState, rand: RandomFn): SpinOutcome {
  const bj = state.blackjackReel!
  if (bj.phase !== 'spinning') throw new Error(`${def.id}: stopReel in phase ${bj.phase}`)
  const r = bj.idx
  const strip = bj.reelStrips[r]!
  const sym = strip[Math.floor(rand() * strip.length)]!
  bj.landed[r] = sym
  bj.idx = r + 1
  if (sym === def.bustSymbol) {
    bj.busted = true; bj.bustBySymbol = true; bj.phase = 'resolved'
    return outcome(def, bj, bj.ante, 0, [{ type: 'bust', reel: r, bySymbol: true }])
  }
  bj.hand.push(sym)
  const acc = freshAcc(); for (const s of bj.hand) applySymbol(acc, def, s)
  const bt = bestTotal(acc.hard, acc.aces)
  bj.hard = acc.hard; bj.aces = acc.aces; bj.multSum = acc.multSum
  if (bt.total > 21) {
    bj.busted = true; bj.phase = 'resolved'
    return outcome(def, bj, bj.ante, 0, [{ type: 'bust', reel: r, bySymbol: false }])
  }
  if (bt.total > bj.bestTotal) bj.bestTotal = bt.total
  if (r === 1 && bt.total === 21) bj.natural = true
  if (bj.idx >= 5) { // survived all five → Five-Card Charlie
    bj.charlie = true; bj.phase = 'resolved'
    const pay = handPayout(def, bj)
    return outcome(def, bj, bj.ante, 0, [{ type: 'charlie', cards: [...bj.hand] }], pay)
  }
  return outcome(def, bj, bj.ante, 0, [{ type: 'reel-stopped', reel: r, symbol: sym }])
}

export function cashOut(def: BlackjackReelMachineDef, state: MachineSessionState): SpinOutcome {
  const bj = state.blackjackReel!
  if (bj.phase !== 'spinning') throw new Error(`${def.id}: cashOut in phase ${bj.phase}`)
  bj.phase = 'resolved'
  const pay = handPayout(def, bj)
  return outcome(def, bj, bj.ante, 0, [{ type: 'cash-out', bestTotal: bj.bestTotal, payout: pay }], pay)
}
```

### Task 5: Deck-depletion exact-RTP DP (optimal stopping)

**Files:** `app/engine/blackjackReelRtp.ts`. Test: `tests/blackjackReelRtp.test.ts`.

Model: at reel `r`, each special lands with prob `count(special in reels[r]) / len(reels[r])`; a card lands with prob `cardSlots(r)/len(r)`, and the card is drawn from the **remaining deck** collapsed to counts by value (`A`, `2..9`, `T`=ten-group of 16). The deck starts `{A:4, 2:4,…,9:4, T:16}`. Cards drawn ≤5, so reachable deck states are bounded → exact enumeration. `EV(cash)=handPayout-without-ante` of the current best total/mult; `EV(continue)=Σ outcomes p·EV(next)`; node = `max`. Reels 1–2 have no cash decision (you always have <2 cards’ worth — but model uniformly: cash on a sub-2-card hand pays 0, so continue dominates). Boundary: reel 5 resolve (Charlie floor + ×).

- [ ] **Step 1: Failing tests** on tiny tuned defs: (a) a def where a near-certain over-21 on the next card makes `optimalStop` return `'cash'` on a qualifying high total; (b) a def where a no-bust multiplier reel makes `'continue'` optimal; (c) determinism (same def → same policy/value); (d) a **closed-form check**: a 1-card-then-forced-cash micro-def whose RTP you can compute by hand matches `blackjackReelExactRtp(def).rtpPerCoin` to 1e-9.

- [ ] **Step 2: Run → fails.**

- [ ] **Step 3: Implement.** State `{ reel, hard, aces, multSum, bestTotal, natural, deck }` where `deck` is the value-count vector; key by a compact string. Recursion mirrors the existing `blackjackReelRtp.ts` `makeSolver` shape (memoized `solve` for value/action + `dist`/`labelled` for the outcome distribution and breakdown buckets `total-N` / `charlie` / `bust`), but:
  - per-reel outcome set comes from `reelOutcomes(def, r, deck)`: specials at fixed probs + a card branch per remaining value `v` at prob `(cardSlots/len) × deck[v]/Σdeck`;
  - applying a value card decrements `deck[v]`, updates `hard/aces`, recomputes `bestTotal` (ratchet) and `natural` (reel index 1, total 21);
  - applying a minus lowers `hard` (floor 0); a multiplier raises `multSum`; the bust symbol or over-21 → terminal 0;
  - `cashValue(s) = base(bestTotal) × max(1,multSum)` with the qualify floor (0 sub-15); reel-5 resolve uses the Charlie floor + `charlieMultiplier`.
  Export `blackjackReelExactRtp(def, opts): ExactRtpReport` (rtpPerCoin = EV(initial empty hand, full deck, reel 0); hitFrequency = P(payout>0); variancePerCoin from the outcome distribution; breakdown buckets), and helpers `optimalStop(def, bj): 'cash' | 'continue'`, `decisionEvs(def, bj)`, `strategyMatrixCell(def, total, reel, opts)` — all built on the same memoized solver so the DP, simulate, X-ray and PAR can never diverge (the project's single-source-of-truth rule).
